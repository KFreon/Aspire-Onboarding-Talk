# Aspire Onboarding and Documentation Talk

# Aspire readme
## Requirements
- dotnet cli
- nodejs

## How to run  
- `dotnet run AspireProj.AppHost/AspireProj.AppHost.csproj`
or F5 in Visual Studio/Rider
- Use buttons to create/seed db and run apps

# Non-Aspire readme
## Requirements  
- dotnet cli
- nodejs
- Sql server
- Postgres
- Seq
- SSMS (or dbgate)
- pgadmin

## How to run  
- Create and seed the databases
- F5 App1 and/or App2
- npm run dev App1.Frontend or App2.Frontend

Ensure ports are setup correctly.
Ensure connection strings are correct across apps.
